IMAGES FOLDER
=============
This project uses high-quality images from Unsplash CDN (loaded via URL in the HTML/CSS).
No local images are required to run the site.

If you want to use local images instead, replace the Unsplash URLs in index.html and style.css
with paths like: images/hero.jpg, images/about.jpg, etc.

Recommended free sources:
- https://unsplash.com (search: golf, golf course, green)
- https://pexels.com
